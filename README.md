# ProCups Ecommerce

Website estatico para a marca ProCups, com loja, promocao 2 por 1, carrinho local e dashboard de gestao estilo Shopify.

## Como abrir

Abre `index.html` no browser.

## Inclui

- Loja com produtos ProCup PSG e ProCup Vasco da Gama.
- Campanha 2 por 1: adiciona 2 unidades e cobra 1 holder de 29,90 EUR.
- Carrinho com persistencia em `localStorage`.
- Checkout simulado.
- Dashboard com visao geral, encomendas, produtos, clientes e definicoes.
- Fluxo operacional de encomendas: Novo, Preparacao e Enviado.

## Proximas ligacoes naturais

- Pagamentos com Stripe, MB Way ou outro gateway.
- Base de dados para clientes, produtos e encomendas.
- Area de login para administradores.
- Modulo futuro para holders de 2 comandos.
